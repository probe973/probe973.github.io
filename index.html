Statistics Workshops

The Learning Enhancement Team

Here you will find a series of guides for using software for statistical analyses. The guides are available for jamovi, R Studio, SPSS, and Stata. There are also guides for using prompts to carry out statistical analysis using AI chat engines.

Each of the guides make use of a fictional dataset. The fictional dataset has 100 entries and contains the following variables:
